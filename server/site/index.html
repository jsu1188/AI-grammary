CW Checkword Writing Assistant
기능 흐름 다운로드 업데이트

Desktop Writing Utility

문서 작성 흐름을 끊지 않고
읽고, 고치고, 다시 적용합니다.

Checkword는 워드, 한글, 브라우저 입력 환경에서 텍스트를 읽고 맞춤법 교정, 요약, 문체 변경, 평가, 제목 추천 기능을 제공하는 데스크톱 도우미입니다. 결과 확인에서 끝나지 않고 실제 문서 작업에 다시 반영하는 흐름을 중심으로 설계했습니다.

  • 워드, 한글, 브라우저 입력 지원

  • 앱 내부 업데이트 알림 연동

Core Features

문서 작성에 필요한 기능을 한 흐름으로 묶었습니다

맞춤법 교정

실시간으로 읽은 텍스트를 분석하고 교정 결과를 바로 확인할 수 있습니다.

서식 보존 적용

가능한 범위에서 기존 문서의 흐름을 유지하면서 교정 텍스트를 다시 적용합니다.

요약과 제목 추천

짧게, bullet, 자세히 요약하고 문서 제목 후보를 함께 제안합니다.

문체/말투 변경

원하는 분위기나 표현 방식으로 문장을 다시 써서 비교할 수 있습니다.

평가와 이유

문장의 점수뿐 아니라 어떤 이유로 그런 평가가 나왔는지도 볼 수 있습니다.

업데이트 알림

앱 시작 시점과 사용 중 주기 체크를 통해 새 버전이 나오면 바로 안내합니다.

Workflow

실제 사용 흐름을 기준으로 설계했습니다

01

텍스트 감지

클립보드, 선택 영역, 실시간 리드 방식으로 현재 작업 중인 텍스트를 가져옵니다.

02

AI 분석

맞춤법, 요약, 문체, 평가, 제목 추천 중 원하는 기능을 바로 실행합니다.

03

결과 적용

결과를 보고 복사하거나 원본 수정으로 이어가며 실제 문서 작업에 반영합니다.
